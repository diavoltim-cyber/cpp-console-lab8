## Описание
Консольное приложение на языке C++ для обработки данных о курсах валют в банках.  
Программа реализует фильтрацию и сортировку информации по заданным условиям.

## Задание
Входные данные:  
- **банк** — строка без пробелов (название банка)  
- **покупка, продажа** — дробные неотрицательные числа  
- **адрес** — строка, может содержать пробелы  

Необходимо:
1. Вывести курсы валюты во всех отделениях банка **«Беларусбанк»** (с адресами).  
2. Вывести курсы валюты и адреса отделений банков, где **продажа < 2.5**.  
3. Реализовать сортировки:
   - **Bubble sort** — по убыванию разницы `(продажа - покупка)`  
   - **Merge sort** — по возрастанию названия банка, а внутри банка — по возрастанию адреса
