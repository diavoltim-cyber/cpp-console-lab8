# Лабораторная работа №8

## Описание
Консольное приложение на языке C++ для выполнения заданий варианта 8.  
Программа демонстрирует базовые операции ввода/вывода и работу с алгоритмами.

## Задание
Реализовать программу, которая:
- Выполняет обработку входных данных в соответствии с вариантом 8.
- Выводит результаты в консоль в удобочитаемом формате.
- Обеспечивает корректную работу как для правильных, так и для некорректных входных данных.
